FSSearchIndexFX by prahaladd

A full text desktop search engine framework written using .NET

he FSSearchIndex Framework project provides a framework that allows application developers to write their own full text content based file search and indexing applications. The framework is very simple to use and a complete file system indexing and searching solution can be written using this framework with very little effort. The framework currently supports the following features: 1.) Full text Content based searching. 2.) Full text Content indexing. 3.) Support for searching and indexing through text content files, MS Word 2007 files, Excel workbooks and PDF files.
